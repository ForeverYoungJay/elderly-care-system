# Implementation Plan and Milestones

## Phase 0 - Foundations (Week 1-2)
- Project scaffolding, CI/CD, base auth
- Core entities: org, staff, resident, bed
- RBAC and permissions

## Phase 1 - Safety + Nursing (Week 3-6)
- Safety plan/task/incident/work order
- Nursing service items, tasks, records
- Attendance and leave

## Phase 2 - Medical + Diet (Week 7-10)
- Medical records, evaluations, vitals
- Monthly checks and reports
- Diet menus, special diets, dispatch

## Phase 3 - External + Reports (Week 11-12)
- External report jobs and receipts
- Dashboards and export center

## Phase 4 - Hardening (Week 13-14)
- Performance tuning, security review
- DR/backups validation
- UAT and release

## Deliverables per phase
- Backend APIs
- Frontend pages
- Data migration/seed
- Automated tests
